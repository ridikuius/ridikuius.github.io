##正则表达式

```java

def reg = "BW -*(\\d+\\.*\\d*)\\s*Kbit[\\s\\S\\r\\n]+10 seconds input rate -*(\\d+\\.*\\d*)\\s*bits/sec.*[^.]+10 seconds output rate -*(\\d+\\.*\\d*)\\s*bits\\/sec"

```

##多行匹配：
```java

def m = s=~ reg
```
