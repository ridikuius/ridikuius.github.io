# 安装 Jekyll

`sudo apt-get update`

`gem sources --removed https://rubygems.org/`

`gem sources -a https://ruby.taobao.org/`

`sudo apt-get install ruby-dev`

`sudo gem install jekyll`

`sudo apt-get install git`

`jekyll new my-awesome-site`

`cd my-awesome-site`

`jekyll serve`
